What is FTC?
       FIRST Tech Challenge, or FTC, is a high school level competition created to inspire young minds around the world to design and build robots that will compete in different competitions. In September, teams worldwide pick up the game manual and start working. Given a set list of parts, teams are limited to using only certain materials for their robot, making the game harder and more challenging.
